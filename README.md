# trueGlass
---

**trueGlass** is a custom theme for Flow Launcher featuring a glass/transparent effect inspired by macOS, minus the blur.

## Features
---
* **Transparent Background:** Clean, glass-like aesthetic.
* **Curved Corners:** Modern UI feel.

## Installation

### Using Terminal (PowerShell/Bash)

```powershell
cd ~/AppData/Roaming/FlowLauncher/Themes
curl -o trueGlass.xaml https://raw.githubusercontent.com/swirik/trueGlass-theme/main/trueGlass.xaml
