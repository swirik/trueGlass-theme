# trueGlass
---

**trueGlass** is my custom theme for flow launcher with glass/transparent effect inspired by macOS glass without the blur.

## What it looks like
---
![[Screenshot 2026-03-08 020602.png]]

## Features
---
- Transparent Background.
- Curved Corners.

## How to Install

> Use bash or powershell.

```powershell
cd ~/AppData/Roaming/FlowLauncher/Themes
curl -o trueGlass.xaml https://raw.githubusercontent.com/swirik/trueGlass-theme/main/trueGlass.xaml
```

> Go to Flow Launcher setting and select trueGlass as your theme.
